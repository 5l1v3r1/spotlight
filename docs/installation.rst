Installation
============

Dependencies
~~~~~~~~~~~~

Spotlight has several external dependencies for diffraction model generation and parallelization.
This includes

* `GSAS <https://subversion.xray.aps.anl.gov/trac/EXPGUI>`_ (rev1253)
* `gsaslanguage <https://github.com/Svennito/gsaslanguage>`_ (@8f0975)
* `OpenMPI <https://www.open-mpi.org/>`_ (v2.3.1)

Furthermore, Spotlight has several Python dependencies which are listed in the ``requirements.txt`` file in the top-level of the repository.
